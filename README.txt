==Description==

THE MOST ADVANCED CONTACT FORM 7 SAVE TO DATABASE PLUGIN.

"CF7NXT Lite" Plugin has designed for saving contact form 7 plugin data into wordpress system so WordPress Users access all data with attachments download in backend with some additional features to manage more flexibly.


== CF7LITE Lite - Awesome Features==

* Save All Contact Form 7 Fields Data
* Save Selected Fields Only
* Label Your Enquiries Like Clients, Sales, Support... etc
* Download Attachments Directly From Panel
* Export All Enquires in CSV Format
* Sort & Export Selected Enquires
* Sort Enquires Via Labels
* Sort Enquires Via Name & Email
* Sort Enquires Via Date
* Starred Important Enquires
* Notification of New Enquiry & More

==Installation==

1. Upload plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, Get "CFT7NXT Lite" Menu



==Frequently Asked Questions==

= Can i store multi selection fields ? =

Yes this plugin also helps you to store fields like multiple checkbox, multiselect etc.

= How many labels i can create ? =

You can create unlimited lables and mark to enquiries, so can sort data & export it easily.
